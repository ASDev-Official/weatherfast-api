WeatherFast APIs

Serverless proxy endpoints that forward requests to data.gov.sg and return the upstream response body as-is.

Routes

GET https://api.weatherfast.aadish.dev/api/sg/two-hour-forecast
GET https://api.weatherfast.aadish.dev/api/sg/twenty-four-hr-forecast
GET https://api.weatherfast.aadish.dev/api/sg/four-day-forecast
GET https://api.weatherfast.aadish.dev/api/sg/air-temperature
GET https://api.weatherfast.aadish.dev/api/sg/rainfall
GET https://api.weatherfast.aadish.dev/api/sg/relative-humidity
GET https://api.weatherfast.aadish.dev/api/sg/wind-speed
GET https://api.weatherfast.aadish.dev/api/sg/psi

Usage

Make a simple GET request to any of the routes above.

Notes

  • Only GET requests are allowed; other methods return 405.
  • Responses and status codes from data.gov.sg are forwarded as-is.
