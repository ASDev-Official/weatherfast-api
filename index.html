WeatherFast APIs

Singapore weather data, proxied securely.

This backend exposes a small set of Singapore weather endpoints that forward requests to data.gov.sg using your server-side API key and return the upstream response body as-is.

Vercel serverless Server-side API key GET only No client secret exposure

Set DATA_GOV_SG_API_KEY in your deployment environment before calling any Singapore route.

Available routes

/api/sg/two-hour-forecast Proxies the 2-hour forecast data.
/api/sg/twenty-four-hr-forecast Proxies the 24-hour forecast data.
/api/sg/four-day-forecast Proxies the 4-day forecast data.
/api/sg/air-temperature Proxies the air temperature data.
/api/sg/relative-humidity Proxies the relative humidity data.
/api/sg/wind-speed Proxies the wind speed data.
/api/sg/psi Proxies the PSI data.
Non-API routes are rewritten to this page. API responses preserve the upstream body and status code.
